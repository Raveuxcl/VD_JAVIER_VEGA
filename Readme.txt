Este proyecto para el curso de visualización de datos se centró en el análisis de relación entre el indicador de calidad de red (específicamente del servicio móvil) UPTIME y los hechos delictuales registrados para la región metropolitana según la CEAD.

Todo el desarrollo, contextualización y análisis en jupyter notebook "VD_JAVIER_VEGA.FIN.ipynb"

Ya que algunas visualizaciones (sobretodo las que se refieren a mapas y algunos módulos de georeferenciación que se fueron probando) podrían no verse en el notebook, se adjunta archivo HTML VD_JAVIER_VEGA.FIN.html de del mismo para que pueda ser descargado y así poder veer todo el recorrido de imagenes, aparte de facilitar la lectura..

El entregable final del proyecto es la infografía:

- Infografía_Javier_Vega_P_FINAL.pdf
